# Phase 1 -- UI Design Contract

> Visual and interaction contract for the Foundation and Design System phase. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | shadcn/ui (new-york style, slate base, CSS variables) |
| Preset | new-york / slate / css-variables |

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Tailwind | Usage |
|-------|-------|----------|-------|
| xs | 4px | `p-1`, `gap-1` | Icon gaps, inline padding, micro spacing |
| sm | 8px | `p-2`, `gap-2` | Compact element spacing, icon button padding |
| md | 16px | `p-4`, `gap-4` | Default element spacing, card padding |
| lg | 24px | `p-6`, `gap-6` | Section padding, card spacing |
| xl | 32px | `p-8`, `gap-8` | Layout gaps, major section padding |
| 2xl | 48px | `p-12`, `gap-12` | Major section breaks |
| 3xl | 64px | `p-16`, `gap-16` | Page-level spacing, hero sections |

Exceptions: none

---

## Typography

Declared sizes: 4 roles. Declared weights: 2 (regular 400, semibold 600).

| Role | Size | Weight | Line Height | Letter Spacing | Font | Tailwind |
|------|------|--------|-------------|----------------|------|----------|
| Body | 14px | 400 | 1.5 (relaxed) | normal | Inter | `text-sm font-normal leading-relaxed` |
| Label | 12px | 600 | 1.0 | 0.1em (wider) | Inter | `text-xs font-semibold tracking-wider` |
| Heading | 20px | 600 | 1.2 (snug) | -0.04em (tight) | Inter | `text-xl font-semibold tracking-tight` |
| Display | 36px | 600 | 1.1 | -0.04em (tight) | Inter | `text-4xl font-semibold tracking-tight` |

**Micro text** (not a declared role, used as utility):
- 10px/11px at weight 500 in JetBrains Mono for timestamps, badges, data labels, version tags
- Tailwind: `text-[10px] font-mono font-medium` or `text-[11px] font-mono font-medium`

**Report body text** (Phase 5+, documented for reference):
- 18px at weight 400, line-height 1.8 in Inter for final report content
- Tailwind: `text-lg font-normal leading-[1.8]`

## Color

### Surface Hierarchy (No-Line Rule)

All boundaries via tonal layering. NEVER use 1px solid borders. Ghost borders at 15% opacity only when accessibility requires (inputs, buttons).

| Role | Value | Hex | Usage |
|------|-------|-----|-------|
| Dominant (60%) | Well | `#0e0e10` | Main workspace, inset areas, deepest background |
| Dominant variant | Surface | `#131315` | Base canvas, page background |
| Secondary (30%) | Deck | `#1c1b1d` | Navigation, sidebars, input backgrounds |
| Secondary variant | Sheet | `#201f22` | Content modules, cards, glass panels |
| Secondary raised | Raised | `#2a2a2c` | Interactive elements, hover states |
| Accent surface | Float | `#353437` | Modals, overlays, floating UI |
| Accent bright | Bright | `#39393b` | Bright accent surfaces, hover lift |

### Semantic Colors

| Role | Value | Usage |
|------|-------|-------|
| Primary | `#c0c1ff` | Primary actions, links, active states, AI Pulse |
| Primary deep | `#8083ff` | Gradient end, hover states, active indicators |
| Primary seed | `#6366f1` | Indigo seed, shadow glow |
| Secondary | `#adc6ff` | Secondary actions |
| Tertiary | `#d0bcff` | Accent highlights |
| On surface | `#e5e1e4` | Primary text -- NEVER pure white |
| On surface variant | `#c7c4d7` | Secondary text, labels, descriptions |
| Outline | `#908fa0` | Subtle separators |
| Outline ghost | `rgba(70, 69, 84, 0.15)` | Ghost borders (inputs, buttons) |
| Error | `#ffb4ab` | Error text |
| Error background | `#93000a` | Error backgrounds |

Accent reserved for: Primary CTA buttons (gradient), AI Pulse indicators, active research step indicators, citation superscripts, primary links, focus rings.

### shadcn CSS Variable Mapping

Override all shadcn HSL CSS variables to Obsidian Deep values. Map to `:root` (dark only, no `.dark` toggle needed):

```
--background: #0e0e10        (Well)
--foreground: #e5e1e4        (on_surface)
--card: #201f22              (Sheet)
--card-foreground: #e5e1e4   (on_surface)
--popover: #353437           (Float)
--popover-foreground: #e5e1e4
--primary: #c0c1ff           (primary)
--primary-foreground: #1000a9
--secondary: #2a2a2c         (Raised)
--secondary-foreground: #e5e1e4
--muted: #1c1b1d             (Deck)
--muted-foreground: #c7c4d7  (on_surface_variant)
--accent: #2a2a2c            (Raised)
--accent-foreground: #e5e1e4
--destructive: #ffb4ab       (error)
--destructive-foreground: #93000a
--border: rgba(70, 69, 84, 0.15)  (outline_ghost)
--input: rgba(70, 69, 84, 0.15)   (outline_ghost)
--ring: #c0c1ff              (primary)
--radius: 0.75rem
```

### Signature Patterns

| Pattern | CSS |
|---------|-----|
| Glassmorphism | `background: rgba(32, 31, 34, 0.6); backdrop-filter: blur(20px); border-radius: 0.75rem; box-shadow: 0px 24px 48px -12px rgba(0, 0, 0, 0.5)` |
